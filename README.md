# Kido Teacher

## How do I install these formulae?

`brew install kido/teacher/<formula>`

Or `brew tap kido/teacher` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kido/teacher"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
